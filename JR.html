Answer to 7

  1. Expand using binomial coefficients or by expanding brackets.

Note1. You should be able to switch between the equivalent notations \[ ^{n}C_{r}=\binom{n}{r}=\frac{n!}{r!(n-r)!} \]

Note2. There are lots of helpful identities involving binomial coefficients. The most important for A level is to realise \[ \binom{n}{r}=\binom{n}{n-r} \] To show this, we start on the right-hand side and end up with the left-hand side simply using the definition

\[ \binom{n}{n-r}=\frac{n!}{(n-r)!(n-n+r)!}=\frac{n!}{(n-r)!r!}=\binom{n}{r} \]

Answering the question

Using binomial coefficents.

\[ (a+x)^{3}=\binom{3}{0}a^{3}x^{0}+\binom{3}{1}a^{2}x^{1}+\binom{3}{2}a^{1}x^{2}+\binom{3}{3}a^{0}x^{3} \] First, \[ \binom{3}{0}=\frac{3!}{0!(3-0)!}=\frac{3!}{3!}=1 =\binom{3}{3}, \:\: \textrm{ by the above result (Note2)} \] Similarly,

\[ \binom{3}{1}=\frac{3!}{1!(3-1)!}=\frac{3!}{2!}=3 =\binom{3}{2}, \:\: \textrm{ by the above result (Note2)} \]


Now we can simplify the terms in \((a+x)^{3}\)

\[ (a+x)^{3}=a^{3}+3a^{2}x+3ax^{2}+x^{3} \]


We can also expand the brackets

\[\begin{align} (a+x)^{3}=(a+x)(a+x)^{2}&=(a+x)(a^{2}+2ax+x^{2})\\ &=(a^{3}+2a^{2}x+ax^{2}+a^{2}x+2ax^{2}+x^{3})\\ &=a^{3}+3a^{2}x+3ax^{2}+x^{3} \end{align}\]


  1. \[\begin{align} \int_{0}^{1}(a+x)^{3}\:dx &=\int_{0}^{1} a^{3}+3a^{2}x+3ax^{2}+x^{3}\:dx\\ &=\left[a^{3}x+3a^{2}\frac{x^{2}}{2}+3a\frac{x^{3}}{3}+\frac{x^{4}}{4}\right]^{1}_{0}\\ &=\left(a^{3}+3a^{2}\frac{1}{2}+3a\frac{1}{3}+\frac{1}{4}\right)-0\\ &=\left(a^{3}+\frac{3}{2}a^{2}+a+\frac{1}{4}\right) \end{align}\]


  1. Now we can solve the equation given as

\[ \int_{0}^{1}(a+x)^{3}=\frac{1}{4}+\frac{a}{2} \] Replacing the left-hand side \[ a^{3}+\frac{3}{2}a^{2}+a+\frac{1}{4}=\frac{1}{4}+\frac{a}{2}\\ \implies a^{3}+\frac{3}{2}a^{2}+\frac{a}{2}=0\\ \implies a \left( a^{2}+\frac{3}{2}a+\frac{1}{2}\right)=0 \] So \(a=0\), or \[ a^{2}+\frac{3}{2}a+\frac{1}{2}=0\\ \]

Completing the square is probably the easiest method to use on this quadratic \[ a^{2}+\frac{3}{2}a+\frac{1}{2}=(a+\frac{3}{4})^{2}-\frac{9}{16}+\frac{8}{16}=0\\ \implies (a+\frac{3}{4})^{2}=\frac{1}{16}\\ \implies (a+\frac{3}{4})=\pm\sqrt{\frac{1}{16}}=\pm\frac{\sqrt{1}}{\sqrt{16}}=\pm\frac{1}{4}\\ \implies a=-\frac{3}{4}\pm\frac{1}{4}=\frac{-3\pm 1}{4} \] So \(a=-1\) and \(a=-1/2\) from the quadratic. Together with \(a=0\), these are the three possible values for \(a\).

Fractions

Here are some exercises to help cement the basics of indices and fractions. These are important as all C1/C2 questions depend upon a fluent handling of this area.

  1. When we write \[ \frac{x}{4} \]

We can also write this, using the same multiplication shorthand as learnt at GCSE. \[ \frac{x}{4}=\frac{1}{4} \times x=\frac{1}{4}(x)=\frac{1}{4}x \] This is so, because of the laws of multiplying fractions. In general

\[ \frac{a}{b} \times \frac{x}{y}=\frac{a \times x}{b \times y}=\frac{ax}{by} \]

Then, working backwards we can take \(a=1\), \(b=4\) and \(y=1\) and we obtain \[ \frac{x}{4}=\frac{1}{4}\frac{x}{1}=\frac{1}{4}x \]


  1. When we write \[ \frac{1}{\frac{2}{3}} \]

We should immediately think this is very horrible. We should always try to simplify these sorts of fraction (in general avoid having more than one dividing bar in a fraction…). This situation sometimes arises in integration.

I will use algebra to show what is going on. We want to express this horrible fraction as a proper fraction. That is, as two numbers \(a\) and \(b\) such that

\[ \frac{1}{\frac{2}{3}}=\frac{a}{b} \]

So now we can manipulate this. First multiply both sides by \(\frac{2}{3}\). This cancels the fraction on the left-hand side

\[ 1=\frac{a}{b}\frac{2}{3}=\frac{2a}{3b} \quad \textrm{using the multiplication rule 1, above} \]


Now multiply both sides by 3 \[ 3 \times 1 = \frac{2a}{3b}\times 3 =\frac{2a \times 3}{3b}=\frac{2a}{b} \]

Now divide both sides by 2 \[ \frac{3 \times 1 }{2}=\frac{2a}{2b}=\frac{a}{b} \]

And there is our answer (\(a=3\) and \(b=2\)). So

\[ \frac{1}{\frac{2}{3}}=\frac{3}{2} \]

In general \[ \frac{x}{\frac{y}{z}}=x\frac{z}{y} \]

All the rules above carry through regardless if fractions are negative

  1. Remember:

\[ -\frac{a}{b}=\frac{-a}{b}=\frac{a}{-b}=-a\frac{1}{b}=a\frac{-1}{b} \] All mean the same thing. We change where we put the minus sign, only to make cancellation clearer \[ \frac{-a}{-b}=\frac{a}{b} \quad \textrm{clearer than,} \quad -\frac{a}{-b}=\frac{a}{b} \]

Indices

These are especially important for C1/C2 - more so than at GCSE. It is important to work through textbook exercises for these, if you feel a little shaky on them, as they are so important for derivatives and integration.


  1. For any number \(a\) (positive, negative, fraction, etc), we have:

\[ \left(\frac{x}{y}\right)^{a}=\frac{x^{a}}{y^{a}}, \quad \left(xy \right)^{a}=x^{a}y^{a} \]

For example, we know \(2^{2}=4\). We also know \(8/4=2\). So, \[ 2^{2}=\left(\frac{8}{4}\right)^{2}=\frac{8^{2}}{4^{2}}=\frac{64}{16}=\frac{2\times2\times2\times2\times2\times2}{2\times2\times2\times2}=2\times 2=4 \]

Similarly,

\[ \sqrt{2}=\sqrt{\left(\frac{8}{4}\right)}=\left(\frac{8}{4}\right)^{1/2}=\frac{8^{1/2}}{4^{1/2}}=\frac{(2\times 4)^{1/2}}{4^{1/2}}=\frac{2^{1/2}4^{1/2}}{4^{1/2}}=\frac{\sqrt{2}\sqrt{4}}{\sqrt{4}}=\sqrt{2} \]


  1. Fractions and differing indices

\[ \frac{x^{a}}{x^{b}}=x^{a-b} \]

For example \[ \frac{x^{5}}{x^{2}}=\frac{x \times x \times x\times x\times x }{x \times x}=x \times x \times x=x^{3}=x^{5-2} \]

This also explains negative indices

\[ \frac{x^{2}}{x^{3}}=\frac{x \times x}{x \times x \times x}=\frac{1}{x}=x^{2-3}=x^{-1} \]

and

\[ \frac{x^{2}}{x^{5}}=\frac{x \times x}{x \times x \times x \times x \times x}=\frac{1}{x\times x\times x}=\frac{1}{x^{3}}=x^{2-5}=x^{-3} \]

In general

\[ x^{-a}=\frac{1}{x^{a}} \quad \textrm{and, } \left(\frac{x}{y}\right)^{-a}=\left(\frac{y}{x}\right)^{a} \]

  1. Multiplication of indices and fractional powers

We write \[ x^{a \times b} = x^{ab} \]

The best way to think of this is as \[ x^{ab}=\left(x^{a}\right)^{b}=\left(x^{b}\right)^{a} \] Note we can compute the power in stages, in any order we like.

For a fractional power (or rational power as it is also known), we make use of the multiplication rule. Let \(b\) and \(d\) be whole numbers, so that \(b/d\) is a fraction then

\[ x^{b/d}=(x^b)^{1/d}=\sqrt[d]{x^{b}} \]

Or we can this the other way around, too

\[ x^{b/d}=(x^{1/d})^{b}=\left(\sqrt[d]{x}\right)^{b} \]

For example \[ 8^{2/3}=(8^{1/3})^{2}=(2)^{2}=4 \] and \[ 8^{2/3}=(8^{2})^{1/3}=(64)^{1/3}=\sqrt[3]{64}=4 \]

Notice how performing \(8^{1/3}\) first is easier than performing \(8^{2}\) first. This is often what guides us when deciding what operation to perform first.
